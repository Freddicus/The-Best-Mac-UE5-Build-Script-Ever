# Code of Conduct

**Don't be a jerk.**

More precisely:

- Disagree about code, not about people. "This approach has a problem" is fine. "You're an idiot" is not.
- Be direct. Bluntness is welcome; cruelty is not. These are not the same thing.
- If someone's approach is wrong, explain why. If it's just different from yours, that's not the same thing.
- Assume good faith until clearly demonstrated otherwise.

Maintainers will close issues or PRs that are rude, exhausting, or off-topic without further explanation. Life is short.
